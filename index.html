Software, from the ground up.

Modern software is built on 50 years of layered, faulty abstraction. We can do better.

Ground is a work-in-progress to build a new verifiable, security oriented software stack from the 'ground up'. The core project consists of a low-level programming language with modern affordances, a self-bootstrapping compiler toolchain, and an airgap-oriented minimal operating system with auditability as its highest priority.

The project also includes computing-educational material. The ultimate goal is the realization of the long-neglected user-developer ethos of FOSS.
